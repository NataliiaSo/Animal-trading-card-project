# Animal Info Card

This repository contains a simple project to create a card that displays information about an animal. The design follows specific layout and style rules, as described below.

## Features

- **Card Layout**: The card displays an animal’s name, image, interesting facts, and a brief description.
- **Fixed Width**: The card has a fixed width of 300px (plus padding on both sides) and does not expand with the browser window.
- **Borders and Spacing**: Borders surround the animal’s name, image, and information sections. Appropriate spacing is applied to separate these elements.
- **Custom Styling**: The card's background colors and border-radius can be customized.
- **Text Styling**: 
  - The interesting fact is *italicized*.
  - The labels for list items are **bolded**.
  - List item dots are removed for a clean look.
- **Image Customization**: The animal image has a fixed width of 300px, with a relevant `alt` attribute for accessibility.

## Project Structure

The project follows best practices for code separation and readability.

- **HTML**: Contains semantic tags and meaningful class names for styling purposes.
- **CSS**: A separate CSS file is linked to the HTML, ensuring a clear separation of concerns.

## Technologies Used

- **HTML5**: For structuring the content.
- **CSS3**: For styling the layout and appearance of the card.
